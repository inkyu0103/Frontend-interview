# CORS

1. 정의

   추가 HTTP 헤더를 사용하여, 한 출처에서 실행 중인 웹 애플리케이션이 다른 출처의 선택한 자원에 접근할 수 있는 권한을 부여하도록 알려주는 체제.
   (다른 출처의 자원에 접근 가능하게 해주는 설정)

2. 동일 출처 정책 (same origin policy)

   두 URL의 `프로토콜`, `포트`, `호스트`가 모두 같아야 동일한 출처이다.
